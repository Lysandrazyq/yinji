# 更新日志 · CHANGELOG

> App Store 风格,只放用户视角条目。详细技术变更看 `devlog/更新日志-开发版.md`。
>
> 版本规则:[SemVer](https://semver.org/lang/zh-CN/) —— `主版本.次版本.补丁版本`。
> 主版本:大改 / 路线调整;次版本:加功能;补丁:修 bug。

---

## v2.9.0 — 2026-05-31

- 新增 **独立的「设置」页**——右上角多了个齿轮图标,点开把所有可调的东西集中放一处,不用再到处找开关
- 新增 **深色模式**——设置里选「跟随系统 / 浅色 / 深色」。跟随系统时,你手机白天浅色、晚上深色,印迹自动跟着换(只影响 App 界面,水印照片本身不变)
- 新增 **一堆贴心设置**:
  - **打开照片时**:要不要记住上次选择、默认用手动还是自动文字、默认水印文字(今天日期 / 留空 / 固定写一句)
  - **自动模式显示哪些信息**:设备、时间、光圈、快门、ISO、焦段,六项各自可勾掉,只留你想要的
  - **照片与导出**:默认导出画质、导出尺寸上限(省流 / 标准 / 高清)、长按看原图开关
  - **数据与隐私**:一键重置常用栏、清空我的模板、恢复默认设置,旁边写清楚"所有数据只在你本机"
- 新增 **新手引导**——第一次打开会有 4 张分步卡片,花十几秒讲清楚怎么选图、挑样式、切文字、保存;看过就不再弹,想再看在设置里点「重看新手引导」
- 把原来藏在「更多模板」面板里的「液态玻璃」开关,搬到了设置页的「外观」里,更合理

> 这是 3.0.0 之前的最后一个 2.x 版本;下一个大版本 3.0.0 会以"AI 能力"为主。设置入口在右上角齿轮;深色模式 / 各项默认都在里面调,选完本机记住。

## v2.8.1 — 2026-05-30

- 新增 **自己挑「常用栏」放哪些模板**——主界面顶部那排快捷模板,现在你说了算。点「更多模板」,每个模板右边有颗「☆ 常用」星星,点亮就把它加到顶部快捷栏,再点取消;常用哪几款本机记住,下次打开还在
- 新增 **自定义模板自动进常用栏**——你新建或导入的模板,会自动出现在顶部快捷栏(和以前一样);删除时也会自动从快捷栏移走
- 优化 **「复制图片」按钮位置**——之前它单独占一行,挤在橙色按钮下面有点怪;现在和「下载文件」「关闭」并排成一行,整齐多了

> 怎么挑常用:右上角点「更多模板」→ 看到喜欢的就点它右边的「☆ 常用」(变成「★ 常用」就是已加入)→ 关掉面板,顶部快捷栏就更新了。至少要留一个常用模板,不会被清空。

## v2.8.0 — 2026-05-30

- 新增 **「复制图片」按钮**——做好的水印照片一键复制,直接粘贴进微信、备忘录、文档,不用先存相册再找出来(支持的浏览器才显示这个按钮)
- 新增 **记住上次的选择**——你上次用的水印样式、文字来源(自动/手动)、导出画质,下次打开自动还原,不用每次重选
- 新增 **长按图片看原图**——在编辑页按住预览图不放,临时显示没加水印的原图,松手就回到水印效果,方便对比前后
- 优化 **玻璃款水印的磨砂模糊做实了**——之前在 iPhone 上几乎看不到模糊,现在是实打实的毛玻璃,更接近 iOS 26 液态玻璃质感
- 优化 **顶部手机状态栏颜色和应用统一**——以前顶部一条色块和页面对不上,现在融为一体
- 优化 删掉底部那行"添加到主屏幕"的小字提示,界面更清爽(装桌面的方法在浏览器菜单里,不影响使用)

> 「复制图片」在保存弹窗里,和「保存到相册」「下载文件」并排;长按看原图在编辑页预览图上(图下方有小字提示)。复制功能在 iPhone Safari、电脑 Chrome / Edge 上都能用,个别老浏览器不支持会自动隐藏按钮、不影响其它功能。

> 新模板都在右上角「更多模板」里,按分类点一下或搜「季节」「富士」「玻璃」「生日」都能找到;玻璃款的磨砂模糊在 iPhone Safari 上最佳,老浏览器会自动退化成半透明纯色。

## v2.7.0 — 2026-05-30

- 新增 **「保存到相册」按钮**——iPhone 上点一下弹系统菜单,选「存储图像」直接进相册(不再只能存到「文件」)
- 新增 **导出画质三选一**:标准 / 高清 / 无损(PNG),要最清晰就选无损
- 新增 保存时**自动保留照片的拍摄信息**(机型 / 时间 / 光圈快门 ISO / 焦距等),存进相册后在系统相册里还能看到
- 优化 保存弹窗更顺手(大按钮 + 画质选择);主界面文字更精简;整体配色字体更细腻
- 优化 清理项目里安卓打包相关的过时文件(统一走 PWA,不影响使用)

> ⚠️ 实况照片(Live Photo)无法在网页里导出动态部分——浏览器只给静态那一帧,这是网页 App 的硬限制。导出的是带水印的静态图。

## v2.6.0 — 2026-05-29

- 新增 自定义模板「分享」——在搜索面板里点自己模板上的「分享」,生成一段代码,复制发微信给朋友
- 新增 自定义模板「导入」——搜索面板里点「导入模板」,把朋友发来的代码粘进去就能用,顶部自动加上按钮
