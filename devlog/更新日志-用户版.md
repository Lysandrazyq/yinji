## 2026-06-09（v3.9.0）🫧 液态玻璃可调，强档更像最新 iOS

- 你点的"适配 iOS27 液态玻璃 + 可调"做好了。设置 →「外观」→「**液态玻璃**」，原来只有开 / 关，现在是 **关 / 轻 / 标准 / 强** 四档：
  - **强**：玻璃感最足、更透、高光鳞边更亮——最接近 iOS 26/27 的 Liquid Glass。
  - **标准**：和你以前看到的一样（默认就是它，不会突然变样）。
  - **轻**：淡淡的磨砂。
  - **关**：纯色，不模糊（最省电、老手机最稳）。
- 调一下，**整个界面**（面板 / 弹窗 / 各种芯片 / 输入框）的玻璃感一起变，选好本机记住。
- 只动 **App 界面的外观**，**不影响你导出的照片**（水印成品一点没变）。

### 📱 这版可以试

1. 删图标→Safari 重开→重新「添加到主屏幕」，版本号 **v3.9.0**。
2. 设置（右上齿轮）→「外观」→「液态玻璃」→ 点 **强 / 标准 / 轻 / 关** 来回切，看面板和弹窗的模糊/通透感跟着变。

> 📌 技术上我没另写一套，而是把玻璃的"模糊+饱和+高光"做成**一个可调的强度变量**，四档只是给它不同的值，所以整套 UI 一起变、改动小、稳。浏览器都验过了：四档的模糊量（关 0 / 轻 16 / 标准 44 / 强 64）在真实面板上逐档生效、设置面板不报错、老用户之前"关了玻璃"的设置也平滑迁移过来、零报错。

## 2026-06-09（v3.8.0）🗂️ 批量加水印：一次给一整组照片套同一套效果

- 这是个**大功能**——旅行一整组、带货一批商品图，终于能**一次性批量加水印**了。
