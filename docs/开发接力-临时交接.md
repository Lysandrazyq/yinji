## 二、🚫 红线（破其一 = 白屏 / 隐私事故，绝不可碰）
1. **单文件**：整个 App 就是 `watermark.html` 一个文件（内联 `<script>` / `<style>`）。
2. **零"加载期"外链**：禁止 `<script src=http>` / `<link href=http>` / `@import http` / `url(http…)`——这些会让手机**白屏**（血泪教训）。运行期 fetch / 异步注入（AI 的 `api.openai.com`、百度统计 `hm.baidu.com`）才允许。
3. **照片绝不上传**：全程本机处理。
4. **Claude 绝不代填 API Key / 密钥**（南心自己粘；百度"站点 ID"是公开标识、可写）。
5. **不引任何第三方库**（连 zip 打包都自己手写 store-zip）。
6. **少灰字**：南心不喜欢界面里的灰色说明小字，别乱加。

## 三、🔁 每发一版的固定流程（照做，别漏）
1. 改 `watermark.html`。
2. **自检**：`cd /e/yinji/yinji-project/yinji` → 用 node 抽出那一段内联 `<script>` → `node --check`（查语法）→ grep 确认**加载期无外链**。
3. **浏览器验证**：用 Claude_Preview MCP 起 `yinji-static` → 先清 SW 缓存 + 带 `?v=时间戳` 重载（否则吃旧缓存）→ 用合成照片走真实 `#fileInput` 管线 → 像素级断言 + console 零报错。
4. **版本号**：`APP_VERSION`（html 里）和 `sw.js` 顶部 `CACHE`（格式 `yinji-vNN-日期-vX.Y.Z`）**一起** bump。
5. **文档**：`CHANGELOG.md`（用户视角）+ `devlog/更新日志-开发版.md`（技术细节）+ `devlog/更新日志-用户版.md`（大白话）+ `CLAUDE.md`（"当前版本/本轮新增"上移、旧的降为"上一版/更早"，**注意别留重复的"上一版"标签**）+ `docs/requirements.md`（决策记录）+ `docs/工作交接.md`（给南心 / 管家）。
6. **生成 docx**：`py tools/md2docx.py`。⚠️ **用 `py` 不要 `python`**（python 没装 python-docx）；Bash 默认 cwd 是父目录 `E:\yinji\yinji-project`，跑项目内命令要先 `cd /e/yinji/yinji-project/yinji`。

## 四、🧱 架构速记（改之前先懂，能少踩坑）
- **渲染管线**：`getSize()`（取图 + `gradedBitmap()` 加滤镜）→ `DRAW_DISPATCH[state.style]`（54 款水印各自画）→ `drawStickerLayer()`（贴纸永远画最上层）。导出走 `exportDataURL` / `buildExport`。
- **加一款水印**（最稳的加法）：季节/品牌类只要给 `SEASON`/`BRAND` 加调色板，再在 `TEMPLATES` / `DRAW_DISPATCH`（指向 `drawSeasonStyle`/`drawBrandStyle`）/ `CAT_IDS` 各登记一条即可，**零新绘制代码**。
- **加一款滤镜**：往 `TONE_LIST` + `TONE_PARAMS` 两张表加数据（参数别越界，照现有取值区间）。
- **本地存储键**：`yinji.prefs.v1` / `yinji.settings.v1` / `yinji.customTemplates.v1` / `yinji.filters.v1` / `yinji.styleColors.v1` / `yinji.stylePacks.v1` / `yinji.glass`。
- **分享码**：`YINJI1:` + base64，`doImport` 按 `type` 路由（水印 `yinji-template` / 滤镜 `yinji-filter` / 风格包 `yinji-pack`）。
- **AI**：`window.Yinji.ai`（4 个：端上配色/文案/荐样式 + 联网文案，**默认关 + 当次授权**）。
- **玻璃**：`--glass-blur` 等 CSS 变量驱动，`body[data-glass-level=light/standard/strong]` 调强度，`data-glass="off"` 走纯色。
