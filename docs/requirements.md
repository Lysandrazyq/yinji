# 开发需求

> 记录"印迹"做什么、做到什么程度、以后往哪走。改需求请更新本文件。

## 1. 目标与定位

一个轻巧、简洁、直观的**照片水印网页 App**，面向 iPhone（Safari）。给照片加上类似 OPPO / 徕卡的相机水印，第一版聚焦最简单的"白框黑字"样式。

## 2. 平台与形态

- **核心**:单文件网页 App(`watermark.html`),所有 UI 和逻辑都在里面。
- **唯一上线路径:PWA**。项目自带 `manifest.webmanifest` + 图标 + `sw.js`,**浏览器打开网址即可"安装到桌面 / 主屏"**,Win / 安卓 / iOS / macOS 全平台都能装,体验接近原生 App,**改一处全平台同步**。
- **已弃用的路线**(2026-05-28 v2.3.0 决策):
  - ~~安卓 Capacitor 打包 .apk~~ — 不再做。原因:维护两套发布太重,PWA 在安卓上体验已经足够;材料(签名 / 软著 / 商店审核)对个人非程序员负责人门槛太高。
  - ~~iOS Capacitor 打包 .ipa~~ — 不再做。原因:需要 Mac + 苹果开发者账号 $99/年。
- **保留可选**:Windows Tauri 出 `.exe` 安装包(给不想用浏览器的桌面用户)。优先级低于 PWA。
- **核心原则**:核心 `watermark.html` 是唯一真相源。所有"装上"方式都不动核心。详见 `docs/cross-platform.md`。

## 3. 核心功能（第一版，已完成）

- **选择照片**：从相册选，或直接拍。
- **26 种预设水印样式 + 自定义模板，一键切换**：

  **基础白款**:
  - **经典**(classic):底部白栏 + 黑字,最克制简洁。
  - **拍立得**(polaroid):四周白边,底部更厚,复古文艺。
  - **宝丽来完全款**(polaroidPlus):加厚拍立得 + 米黄底 + 暖色调,做旧。
  - **极简框**(minimalFrame):细白边 + 下方小字,近乎隐形。
  - **杂志浅版**(magazineLight):白底 + 黑字大标题。

  **深色款**:
  - **黑底白字**(darkBar):底栏深黑 + 白字,夜景照片更搭。
  - **电影黑边**(cinema):上下黑色横条 + 白字,16:9 letterbox 感。
  - **杂志封面**(magazine):照片满版 + 底部暗渐变 + 白色大标题,时尚杂志风。
  - **哈苏风**(hasselblad):深色底栏 + 橙色细条 + 橙字。

  **彩色 / 特殊款**:
  - **徕卡**(leica):底栏顶部一道粗徕卡红横条 + 自动模式分隔线变红。
  - **节日红**(festivalRed):红色边框 + 金色文字,过年/节庆。
  - **复古胶片**(filmStrip):左右两侧加胶片齿孔,米黄底,胶卷感。
  - **签到打卡**(checkIn):底栏带红色定位图标,旅行打卡风。
  - **大字日期**(bigDate):照片上压巨大年份数字,大字报感。
  - **极简右下角**(minimalCorner):不加底栏,右下角一行小字。

  **季节 / 节气款**(v2.7.1 新增,一套渲染四色):
  - **春·樱**(springCherry):粉色调底栏 + 细横线,春日感。
  - **夏·海**(summerSea):清爽蓝调,夏天/海边。
  - **秋·枫**(autumnMaple):暖橙调,秋意。
  - **冬·雪**(winterSnow):冷灰调,冬日/雪景。

  **相机品牌风**(v2.7.1 新增,深色底栏 + 品牌色细条 + 品牌色字):
  - **富士绿**(fujiFilm):墨绿底 + 富士绿。
  - **索尼橙**(sonyAlpha):黑底 + 索尼橙。
  - **尼康黄**(nikonGold):暖黑底 + 尼康黄。

  **玻璃质感款**(v2.7.1 新增,半透明磨砂浮在照片上,iPhone Safari 最佳):
  - **玻璃底栏**(glassBar):底部磨砂玻璃横栏 + 白字,液态玻璃感(主样式条直达)。
  - **玻璃卡片**(glassCard):底部居中浮一块圆角磨砂卡片装文字。

  **节日 / 纪念日款**(v2.7.1 新增):
  - **生日快乐**(birthday):暖色渐变底栏 + 彩色纸屑点 + 棕字。
  - **纪念日**(anniversary):白底栏 + 玫瑰色爱心 + 玫瑰字。

- **自定义模板**:点搜索弹窗里的「+ 新建我的模板」 → 选布局(底栏 / 四周白边 / 满版渐变)→ 调底色 / 文字色 / 强调色 → 起名保存。保存在浏览器 localStorage,本机持久;在搜索面板里能找到、能编辑、能删除。
- **两种文字模式，一键切换**（和样式正交，可任意组合）：
  - 手动输入：用户自己写一句话（默认预填当天日期）。
  - 自动读取：从照片 EXIF 读出**拍摄时间、设备型号、光圈、快门、ISO、焦段**，排成相机水印样式。
- **实时预览**：改文字 / 切模式 / 切样式即时更新。
- **保存**：点「保存到相册」直接调系统「存储图像」进相册（不支持时退化为下载 / 长按存图）；可选导出画质（标准 / 高清 / 无损 PNG）；JPEG 导出自动保留原图拍摄信息（EXIF）。
- **模板搜索**：右上角搜索图标点开，弹出输入框 + 模板名单，按名字 / 描述 / 标签实时筛选（如「红」「复古」「杂志」「暖」），点一下应用并关闭。
- **复制到剪贴板**（v2.8.0）：保存弹窗里「复制图片」一键把成图写进系统剪贴板（PNG），直接粘贴到聊天 / 备忘录 / 文档（浏览器支持 `ClipboardItem` 时才显示）。
- **记住上次选择**（v2.8.0）：上次用的水印样式、文字模式（自动 / 手动）、导出画质本机记住，下次打开自动还原，不用重选。
- **长按看原图**（v2.8.0）：编辑时长按预览图临时显示无水印原图，松手恢复，方便对比加水印前后。
- **自挑主样式条常用模板**（v2.8.1）：主界面顶部那排快捷模板，由用户自己挑显示哪些。在「更多模板」面板里点每张卡右侧的「☆ 常用」星标即可加入 / 移出顶部快捷栏，选择本机记住；新建 / 导入的自定义模板自动加入，删除自动移出。不改模板内容，只改"快捷条上放谁"。
- **独立设置页**（v2.9.0）：右上角齿轮图标打开，把所有可调项集中到一处，分六组：
  - **外观**：主题（跟随系统 / 浅色 / 深色）、液态玻璃开关（从「更多模板」面板搬来）。
  - **打开照片时**：记住上次选择开关、默认文字模式（用上次的 / 手动 / 自动）、默认水印文字（今天日期 / 留空 / 固定一句）。
  - **自动模式显示哪些信息**：设备 / 时间 / 光圈 / 快门 / ISO / 焦段 六项各自可勾选，自动模式下只显示勾选的。
  - **照片与导出**：默认导出画质、导出尺寸上限（省流 1600 / 标准 2200 / 高清 3200，指最长边像素）、长按看原图开关。
  - **数据与隐私**：重置常用栏 / 清空我的模板 / 恢复默认设置，附隐私说明。
  - **关于**：版本号、重看新手引导。
  - 所有设置写入 localStorage `yinji.settings.v1`，本机持久。
- **深色模式**（v2.9.0）：设置「外观」里选「跟随系统 / 浅色 / 深色」。跟随系统时随 `prefers-color-scheme` 自动切换。**只影响 App 界面配色，不改水印照片本身的颜色**。
- **新手引导**（v2.9.0）：首次打开自动弹 4 张分步卡片（欢迎 + 本地不上传 → 选图 + 挑样式 → 手动 / 自动文字 → 保存 / 复制），看过本机记住不再弹；设置「关于」里可「重看新手引导」。
- **添加模板按钮 / 默认打开模板 / 文件名 / 新样式**（v2.9.1）：
  - 主样式条最前的「+」改为「＋ 添加模板」，一眼看懂是新建自己的水印款。
  - 设置「打开照片时」加「默认打开模板」下拉：指定每次开图默认套用的样式，或保持「用上次用的」。
  - 导出文件名带日期 + 样式名（如 `印迹_2026-05-31_经典.jpg`），相册 / 文件里好认好归类。
  - 新增 4 款预设（佳能红 / 毕业季 / 莫兰迪 / 节气·青），样式 26 → 30 款。
- **3.0 AI 接口预留**（v2.9.1）：代码层留出 `window.Yinji.ai` 扩展点 + 只读上下文（不含照片像素），2.x 不产生任何 AI 行为、不联网；并在设置加「AI · 即将推出」预告段。为 3.0 接入做准备，详见 `tech.md` 6.11。

## 4. UI 原则

简洁、直观为第一原则。主界面只暴露必要操作（选图 → 选模式 → 保存），不堆功能。具体视觉见 `design.md`。

## 5. 隐私

照片仅在本地处理，**不上传**到任何服务器。

## 6. 已知限制（第一版接受）

- 自动读取依赖照片自带 EXIF；截图、转发过的照片常丢失，读不到时自动切回手动输入并提示。
- 不含地点信息（GPS 转地名需额外服务，且常被系统抹除）。
- iPhone 的 HEIC 格式：拍摄信息读取已在 iPhone Safari 支持（v2.5.0）；非 Safari 浏览器仍可能解不开 HEIC 图像，会提示先转 JPEG。
- 实况照片（Live Photo）只能导出静态那一帧 + 水印，动态部分网页拿不到（浏览器只交付静态封面帧，硬限制）。
- 复制图片到剪贴板依赖浏览器 `ClipboardItem` 写图能力（iPhone Safari、桌面 Chrome / Edge 支持）；个别老浏览器不支持时按钮自动隐藏，仍可用「保存到相册 / 下载文件」。
- 深色模式只换 App 界面配色，不改水印照片输出（水印自带固定配色，按样式来）。「跟随系统」依赖浏览器 `prefers-color-scheme`，极老浏览器不支持时按浅色处理。

## 8. 决策记录（为什么这么定）

- **核心永远是网页**:负责人非程序员,网页零安装、零成本、改一处全平台同步。原生 App 只做"套壳",不是另起炉灶。
- **样式与文字模式正交**:样式控制外观,文字模式(手动 / 自动)控制内容,互不影响,组合更灵活。
- **自定义模板派生**:自定义模板不是从零写,而是基于"布局类型"(底栏 / 白边 / 渐变) + 颜色覆盖。这样实现简单,用户也好上手。
- **默认模式**:有 EXIF 时默认"自动读取"(更出彩),无 EXIF 时默认"手动输入"。
- **平台接口而非平台条件分支**:`window.Yinji.platform.*` 抽象保存 / 选图 / 存储等动作,各平台只是注入不同实现,核心代码不变。
- **设置集中、键分散**(v2.9.0):新增设置统一收进一个 `yinji.settings.v1` 对象(以后加开关便宜),但**不合并**已有的 `yinji.glass` / `yinji.prefs.v1` / `yinji.segPins.v1` 三个老键——各自独立、避免迁移回归。读取用惰性 `ensureSettings()` + "存储值合并到默认值之上",对部分 / 损坏 / 旧数据健壮。
- **深色模式只换界面、不碰水印**(v2.9.0):深色是 `body[data-theme]` 上的一层 UI 调色板覆盖,canvas 水印输出完全不受影响(水印颜色按样式固定)。"跟随系统"在 JS 里用 `matchMedia` 解析成显式浅 / 深,CSS 只认 `light/dark`。
- **新手引导首跑一次**(v2.9.0):用 `yinji.onboarded.v1` 标记是否看过,首跑自动弹、看过不再扰;设置里可重看。分步卡片(而非一次性长文)更轻、可跳过。
- **3.0 AI 接口提前预留**(v2.9.1):趁 2.x 收尾把扩展点定下来,避免 3.0 临时改核心结构。做法——在 `window.Yinji` 下加 `ai` 命名空间(与既有 `platform` 平级,`register/list/run`)+ `getContext()` 只读快照;快照**只给元信息(版本 / 有无照片 / 样式 / 模式 / 文字),绝不含照片像素**。2.x 不注册任何 AI 提供方,`run` 是空操作、不联网、零行为变化。隐私底线写进 `tech.md` 6.11,3.0 据此实现不得漂移。
- **新预设走"调色板 + 共享渲染器"**(v2.9.1):4 款新样式(佳能红 / 毕业季 / 莫兰迪 / 节气·青)不写新绘制代码,只给两个已验证的共享渲染器(`drawSeasonStyle` / `drawBrandStyle`)加调色板再登记,改动小、回归风险低。
- **常用栏从"预置"改"自挑"**(v2.9.2):顶部样式条默认空,首屏只留一个醒目的大「添加模板」按钮,把"放哪些快捷样式"完全交给用户(在「更多模板」点「☆ 常用」)。理由:预置一排样式对新人是噪音、也不暗示"可换",而空栏 + 一个大按钮更聚焦"先建 / 先挑";要快捷栏的人自己钉,所见即所选。代价:存储键 `segPins.v1→v2` 让新默认对老用户立即生效,旧的自动预置会清一次(秒级可重钉,自定义模板 / 照片不受影响)。配套**取消"至少留一个"守卫**,使"默认空"与"可清空到 0"自洽。
- **调色在 `getSize()` 单点注入、用像素而非滤镜**(v2.12.0):把"调一次色 → 缓存离屏 canvas → `getSize()` 返回它当 `bmp`"放在所有绘制 / 导出的上游,使 35 款样式 + 导出零改动全继承调色,而非逐样式接调色参数。颜色变换走 `getImageData`/`putImageData` 逐像素算——因为 iOS Safari 对 canvas 不应用 `ctx.filter` 的颜色类滤镜(同玻璃磨砂踩过的坑)。Beta 只给现成色调,强度可调等留 3.0,避免 2.0 收官版背太重的新 UI。
