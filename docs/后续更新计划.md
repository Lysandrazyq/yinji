## 二、5.0 候选方向

### 🅰️ 智能增强（AI 支线，3.0 框架已铺好）

- **一句话生成水印样式**：说"我要复古胶片暖调"，AI 帮你挑水印+滤镜+调色盘。接既有 BYOK 通道，默认关+当次授权。
- **智能去背景**：贴纸/Logo 时一键去掉图片白底或纯色背景。纯端上，不联网。

### 🅱️ 协作与分享

- **批量分享**：批量处理完的照片，一键发微信/朋友圈（调用系统分享面板，一次选多张）
- **水印模板市场**（离线版）：生成一个网页链接，别人打开就能预览+下载这套水印配置，不用装 App

### 🅲️ 编辑增强

- **文字样式**：粗体/斜体/字号独立调节（现在字号是全局滑杆，不能单款调）
- **多照片拼接**：把两张照片拼成一张（上下/左右/电影画幅），再加水印
- **贴纸库扩充**：更多内置 emoji 贴纸 + 简单形状（圆/方/箭头/线条）

### 🅳️ 体验打磨

- **历史记录面板**：侧边栏显示最近编辑过的照片（存缩略图，本机 IndexedDB）
- **预设推荐**：打开照片时，根据色调自动推荐 3 款水印+滤镜组合
- **导出尺寸预设**：小图（朋友圈）/ 中图（小红书）/ 原图，一键切换
