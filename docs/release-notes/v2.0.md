## 二、配图建议(挑一种做)

### 方案 1:九宫格对比图(推荐,最有冲击)

挑 9 张你拍过的照片(主题尽量不同,有人像/风景/物件/吃喝/夜景等),每一张应用一个**视觉差异最大**的模板,排成 3×3:

| 第 1 行 | 第 2 行 | 第 3 行 |
|---|---|---|
| 经典 + 日常照 | 徕卡 + 街拍 | 杂志封面 + 人像 |
| 拍立得 + 物件照 | 宝丽来完全款 + 老物件 | 节日红 + 过年照 |
| 电影黑边 + 风景 | 大字日期 + 纪念日 | 复古胶片 + 风格化 |

**制作工具**:微信本身的"九宫格"功能,或用「拼立得」「美图秀秀」「VSCO」拼图。

### 方案 2:横排 4 连图

挑 4 张最具代表性的对比:

1. **经典**(基础款,展示克制美)
2. **徕卡**(红条标志性强)
3. **杂志封面**(满版大标题,最炸)
4. **宝丽来完全款**(米黄复古,文艺感)

**配文**:"15 种风格,4 种代表" / "总有一款适合你"

### 方案 3:单张主图 + 一张界面截图

- 主图:精心选一张照片 + 你最喜欢的模板(推荐宝丽来完全款 / 杂志封面)
- 截图:截一张主界面(显示「样式」选择条 + 自己捏的几个模板按钮 + 搜索图标)

**配文**:"做了个工具,想拍照后给照片打个时间戳一样地用"

### 方案 4:对比 GIF / 视频

录屏:同一张照片,你手指在「样式」选择条上左右滑,15 种模板**实时切换**的动效。

**工具**:手机自带录屏 → 转 GIF(微信不支持长视频原画,GIF 兼容性好)。

## 四、回复评论的话术(常见问题预设)

| 用户问 | 回答 |
|---|---|
| 怎么安装到 iPhone? | Safari 打开网址 → 底部分享 → 添加到主屏 |
| 安卓怎么装? | Chrome / Edge 打开 → 地址栏右边的"安装"图标 |
| 收费吗? | 完全免费,本机处理,代码也开源 |
| 照片会上传吗? | 不会,所有处理都在你浏览器本地,断网也能用 |
| 可以批量加水印吗? | 当前不支持,得一张一张加(后续版本会加) |
| HEIC 格式能读到拍摄信息吗? | 目前 HEIC 拍摄信息读不到,会自动切到手动模式 |
