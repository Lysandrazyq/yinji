## 三、🗂️ 批量加水印（一次处理一组照片）

> v3.8.0

把一张照片调到满意后，点「**🗂️ 把这套效果批量加到多张照片**」→ 选多张 → App 逐张套上**同一套水印 + 滤镜 + 调色 + 主题色 + 贴纸**，最后打包成一个 zip 下载。

- 每张**自动读自己的拍摄信息**（自动模式）
- 一次最多 30 张，有进度条
- **全部在你手机本机处理，照片绝不上传**
- 拿到 zip 后到「文件」App 长按解压

## 四、🎨 我的风格（存 / 套用 / 分享 / 批量联动）

> v3.6.0 + v3.7.0 + v3.13.0

### 存为一键风格
在「水印」页顶部，把当前这套**水印 + 滤镜 + 调色 + 主题色**存成一个命名风格，下次点一下**一键套用**。

### 分享给朋友
选中风格 → 点「**分享**」→ 生成一段离线代码 → 发给朋友 → 对方点「**导入风格**」粘贴 → 整套效果还原。

### 批量时直接用风格包
批量加水印时，按钮上方多了「**批量套用风格**」选择器——点"当前效果"用编辑器这套，点你存的风格包就用那套。

## 五、🚀 PWA 分享目标（从相册一键直达）

> v3.12.0

相册 / 微信 / 任意 App → 点「分享」→ 选「印迹」→ 照片**直达编辑器**，不用先开网页再选图。

- **安卓和电脑 Chrome** 体验最好，分享列表里会出现「印迹」
- iPhone（iOS Safari）支持有限——如果没出现入口，照旧手动选图，不影响任何功能

## 六、🔧 文字微调（栏高 + 字号可调）

> v3.10.0

「水印」页滚到底，多了「**高级微调 ▸**」，点开两个滑杆：

- **栏高**（50% ~ 200%）：调水印底栏的粗细
- **字号**（50% ~ 200%）：调文字的大小

拉了立刻生效、本机记住。「恢复默认」回到 100%。所有水印都支持。

---

## 七、🫧 液态玻璃强度可调

> v3.9.0

设置 →「外观」→「液态玻璃」从开/关升级成 **关 / 轻 / 标准 / 强** 四档。

- **强**档更透、高光更亮，接近 iOS 26 / 27 质感
- **标准**和以前一样
- **轻**更淡
- **关**纯色（最省电）

只改 App 界面外观，不影响照片成品。

---

## 八、🔤 多行水印文字

> v3.4.0

手动输入框支持**按回车换行**，最多 3 行。写地址、写两行诗句、写三个标签——一行一行叠起来，每行按它自己的明暗自动选黑/白字。

---

## 九、🖼️ 自定义 Logo / 图片贴纸

> v3.3.0

贴纸系统现在支持贴**图片**——自己的 Logo、签名图片、小图标，贴上去可以**拖动、缩放**。只在你手机上读，不上传。

## 十、📍 拍摄地点

> v3.1.0

设置里多了一个「**地点**」选项——勾上后，如果照片自带了 GPS 坐标，水印里会显示经纬度；你也可以手打一个地名。**默认关。**

## 十一、🔤 文字颜色自适应 + 按样式记忆调色

> v3.2.0

- **文字自适应**：右下角极简款会根据照片那个角落的明暗，自动选黑字还是白字，保证能看清。
- **按样式记忆调色**：切水印时，每个水印会**记住你上次给它配的滤镜 / 调色**，切回去自动还原。
