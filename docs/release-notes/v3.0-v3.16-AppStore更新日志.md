# 印迹 · 3.0 正式版 更新日志

> 从 v3.0.0 到 v3.16.0，16 个版本的累积更新。

---

## 新功能

- **批量加水印**：调好一张 → 一次套到多张照片 → 逐张存相册或打包下载，全部本机处理不上传
- **AI 智能帮手**：✨ 帮我配色 / 来点灵感 / 帮我选样式 / 联网文案——端上优先、默认关、照片不上传
- **风格包**：把水印+滤镜+调色+主题色存成一套，一键套用、离线分享给朋友
- **PWA 分享目标**：相册点「分享」→ 选「印迹」→ 照片直达编辑器（安卓/桌面体验好）
- **「浏览全部」抽屉**：右侧滑出面板，水印和滤镜一屏看全，还能搜名字
- **液态玻璃可调**：关/轻/标准/强 四档，强档接近 iOS 26/27 质感

## 更多款式

- **水印 95 款**：经典/深色/彩色/季节/品牌/玻璃/节日，应有尽有
- **滤镜 87 款**：胶片/黑白/暖色/冷色/清新/风格，六类全覆盖

## 细节打磨

- 多行水印文字（回车换行，最多 3 行）
- 文字微调（栏高 + 字号滑杆）
- 自定义 Logo / 图片贴纸（可拖动缩放）
- 文字颜色自适应（自动选黑/白）
- 拍摄地点（GPS 坐标 + 手填地名）
- 拼图（2~9 张多布局）+ 贴纸（emoji/手绘/文字/图片）
- 按样式记忆调色（切水印自动还原上次的滤镜）

## 始终不变

- 照片**全程本机处理，绝不上传**
- 打开即用，断网也能用
- 零外部依赖，不偷跑流量
